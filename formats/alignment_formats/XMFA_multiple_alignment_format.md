title: XMFA multiple alignment format
layout: default
---

 A modification of the FASTA multiple alignment format that allows for multiple alignments in a file. More information is on the [MAUVE site](http://asap.ahabs.wisc.edu/mauve-aligner/mauve-user-guide/mauve-output-file-formats.html).

The defline contains the following in its header: 

`>seqname:start-stop strand comments`
