Description
-----------

The PLN format stores a single sequence over one or more lines, with no annotation - it's *plain* !

Example
-------

```
GGGATAGATATATAGCGCGCTCGCCTCGCTCCGCTCTCGCGTC
GTTGTCGCTCTCGCTCGCGCTCTCTCTCCGTCCTTCTTTATATA
GGCTCTCG"

```
